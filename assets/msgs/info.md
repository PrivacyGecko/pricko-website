  1. Privacy-gecko-website (privacygecko.com)

  Current directory | Next.js 14 | Production-ready

  Tech Stack:
  - Framework: Next.js 14 with App Router
  - Language: TypeScript
  - Styling: Tailwind CSS + Framer Motion
  - Payments: Stripe integration
  - Deployment: Vercel

  Structure:
  - 28+ pages including products, pricing, blog, legal, resources
  - Complete marketing site for Privacy Gecko suite
  - 8 privacy tools: GeckoAdvisor, GeckoShare, GeckoGuard, + 5 coming in 2025
  - API routes for checkout, contact, newsletter
  - SEO optimized with sitemap, robots.txt, OpenGraph images

  Status: Fully functional production website

  2. Pricko-Website (pricko.com)

  ../Pricko-Website | React + Vite | Memecoin site

  Tech Stack:
  - Framework: React 18 + Vite
  - Language: TypeScript
  - Styling: Tailwind CSS + Framer Motion
  - Routing: React Router DOM

  Structure:
  - 6 main pages: Home, About, Tools, Tokenomics, Roadmap, Contact
  - Focus on $PRICK token and privacy tools (PrickoVPN, BurnerChat, DoxxGuard)
  - Dark theme with green gecko branding
  - Simpler SPA architecture

  Status: Complete memecoin marketing site

  ---
  Key Differences

  | Aspect    | Privacy Gecko            | Pricko                   |
  |-----------|--------------------------|--------------------------|
  | Framework | Next.js (SSR/SSG)        | React SPA (Vite)         |
  | Scope     | Full SaaS suite          | Memecoin + tools         |
  | Pages     | 28+ pages                | 6 pages                  |
  | Features  | Stripe, Blog, API routes | Token info, simple pages |
  | URL       | privacygecko.com         | pricko.com               |

  Both projects share the gecko branding and privacy focus, with Privacy Gecko being the main product suite and Pricko being the community-driven token side.
